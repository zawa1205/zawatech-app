## Getting Started

First, enable pnpm:

```bash
corepack enable
```

Second, install packages:

```bash
pnpm i
```

### Run Next app on development:

```bash
pnpm dev
```

### Run storybook on development:

```bash
pnpm sb
```

## Build commands:

The built files are not managed in git.

### Build Next app:

```bash
pnpm build
```

### Build storybook:

```bash
pnpm build:sb
```
